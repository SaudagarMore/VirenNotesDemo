â• BONUS: Host Multiple Sites (Virtual Hosting)

1. Create a folder:

```bash
sudo mkdir -p /var/www/mysite.com/html
```

2. Create HTML file:

```bash
echo "<h1>Welcome to mysite.com</h1>" | sudo tee /var/www/mysite.com/html/index.html
```

3. Create config file:

```bash
sudo nano /etc/nginx/sites-available/mysite.com
```

Paste:

```nginx
server {
    listen 80;
    server_name mysite.com www.mysite.com;

    root /var/www/mysite.com/html;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

4. Enable the site:

```bash
sudo ln -s /etc/nginx/sites-available/mysite.com /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

ğŸ¯ You can now map `mysite.com` to your server's IP (via DNS or `/etc/hosts`) and access it.
