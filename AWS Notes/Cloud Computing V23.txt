Traditional Way (Before Cloud)

* Watch movies using DVDs or store files on USBs.
* Buy and manage physical servers (costly, complex, time-consuming).
* Scaling up = more money, time, and headache.
* If traffic drops, unused servers = waste of money.

Cloud Way

* **Access computing resources (servers, storage, apps)** via the Internet.
* Hosted in **remote data centers** (aka "the cloud").
* Pay only for what you use (**pay-as-you-go**).
